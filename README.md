# Regression of Charges

# Project Description

A machine learning project in which an ML model predicts numerical values based on other input features.

# Usage Instructions

1. Clone the repository

git clone https://github.com/FruIT-Prog/Regression-of-charges.git
cd Regression-of-charges

2. Create a virtual environment
python -m venv venv

3. Activate the virtual environment
Windows: venv\Scripts\activate
Linux/MacOS: source venv/bin/activate

4. Install dependencies
pip install -r requirements.txt

5. Run the projects
python main.py
