# Regression of charges

Проект, в котором ML модель предсказывает числовые значения на основе других признаков.

# ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ

1. Клонировать репозиторий
git clone https://github.com/FruIT-Prog/Regression-of-charges.git
cd Regression-of-charges

2. Создать виртуальное окружение
python -m venv venv

3. Активировать окружение

Windows: venv\Scripts\activate

Linux/MacOS: source venv/bin/activate

4. Установить зависимости
pip install -r requirements.txt

5. Запустить файл
python main.py
